# Node.js 中间件

## 1. 什么是中间件

在 Node.js 中，中间件是指在请求-响应周期中执行的函数，可以访问请求对象(req)、响应对象(res)和下一个中间件函数(next)。中间件的主要作用包括：

- 执行任何代码
- 修改请求和响应对象
- 结束请求-响应周期
- 调用堆栈中的下一个中间件
